## Key formulas used
- Inherent score = Likelihood × Impact
- Residual likelihood = MAX(1, Likelihood - ROUND(Control Strength/2, 0))
- Residual impact = MAX(1, Impact - ROUND(Control Strength/3, 0))
- Residual score = Residual likelihood × Residual impact
- KPI formulas include COUNTIF, COUNTIFS, AVERAGEIF, IF, MAX, ROUND, TODAY, and VLOOKUP
